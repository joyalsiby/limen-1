# LIMEN-1

> **"Silence is the most honest form of communication."**

LIMEN-1 is a psychological, curiosity-driven web experience that challenges the necessity of visual noise. It is designed as an abandoned, highly intelligent terminal that observes the observer.

Developed using Vanilla JS and the Web Audio API, the project utilizes progressive disclosure and stillness-based mechanics to bridge the gap between user and interface.

## 👁️ The Concept
LIMEN-1 (Latin for *threshold*) is a space designed for patience. In an era of demanding notifications and saturated interfaces, LIMEN-1 offers a void. It strips away color, shape, and motion until only the core remain.

### Key Mechanics:
- **Stillness-Based Interaction**: Access is granted only to those who can remain absolutely still.
- **Progressive Narrative**: The story of J and the system unfolds through a custom CLI.
- **Dynamic Audio Synthesis**: Real-time Web Audio synthesis mapped to system transitions and terminal events.
- **Observer Effect**: The system tracks and reacts to the user's presence, including tab switches and cursor behavior.

## 🛠️ Technology
- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+).
- **Audio**: Web Audio API for real-time sound synthesis.
- **Typography**: Playfair Display (Serif) and JetBrains Mono (Monospace).
- **Deployment**: GitHub Pages with a custom domain.

## 🚀 Deployment
This project is configured for deployment on **GitHub Pages**.

### Steps to Deploy:
1. **Push to GitHub**: Initialize a repository and push all files to the `main` branch.
2. **Enable Pages**: Go to **Settings > Pages** in your GitHub repository.
3. **Custom Domain**: The included `CNAME` file is configured for `limen1.joyalsiby.com`. Ensure your DNS provider (Cloudflare, Namecheap, etc.) has a CNAME record pointing to `yourusername.github.io`.
4. **HTTPS**: Enforce HTTPS in the GitHub Pages settings for a secure experience.
